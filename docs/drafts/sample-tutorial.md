## Parte 1 - ¿Qué es Vertex AI? (0:30-1:45)
[VISUAL: Diagrama animado de los 5 modelos principales: Veo, Imagen, Chirp, Lyria, Gemini]

[SERIES DE VOZ]:
"Vertex AI es la plataforma de Google para crear contenido usando modelos generativos. Tiene cinco modelos principales:

Primero, Veo, para generar video.

Segundo, Imagen para crear imágenes fotorrealistas.

Tercero, Chirp para narración con múltiples voices.

Cuarto, Lyria para música original.

Y el quinto, Gemini, para crear guiones y razonamiento complejo."

## Parte 2 - Flujo de Producción (1:45-3:30)
[VISUAL: Flujo animado: Guion → Imágenes → Video → Voz → Música → FFmpeg → Video final]

[SERIES DE VOZ]:
"El proceso funciona así. Primero usamos Gemini 2.5 Pro para escribir el guión con indicaciones visuales.

Luego, generamos imágenes con Imagen 4.

Después, usamos Veo 3.1 para crear el video.

Añadimos narración con Chirp 3.

Bajo música con Lyria 2.

Y finalmente, usamos FFmpeg para ensamblar todo en un video completo."
