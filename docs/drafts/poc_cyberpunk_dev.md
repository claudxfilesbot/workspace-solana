# Guion PoC: "The Code Nexus" (Cyberpunk/Tech)

## Concepto
Un viaje visual a través de la creación de software en un futuro donde el código y la realidad se fusionan.

## Formato
- **Aspecto:** 9:16 (Vertical / Reels / TikTok)
- **Duración estimada:** 15-20 segundos
- **Tono:** Épico, tecnológico, futurista.

---

## Escena 1: El Despertar del Código (0-5s)
- **Visual:** Primer plano de ojos reflejando líneas de código Python (funciones, loops) fluyendo a gran velocidad. Luces de neón azules y púrpuras.
- **Prompt Imagen 4:** Close up of human eyes reflecting glowing blue and purple Python source code, recognizable syntax like 'def', 'if', and 'return', cyberpunk aesthetic, cinematic lighting, ultra-realistic, 8k.
- **Prompt Veo 3.1:** The code in the eyes starts to flow faster, the pupil dilates, digital glitch effects.
- **Narración:** "El desarrollo de software ya no es solo escribir líneas..."

## Escena 2: Arquitectura de Datos (5-12s)
- **Visual:** Un programador con interfaz neuronal (o cables sutiles) manipulando una ciudad holográfica hecha de cubos de datos y algoritmos.
- **Prompt Imagen 4:** A futuristic software developer wearing neural interface gear, hands manipulating a complex 3D holographic city made of glowing data blocks and circuit lines, dark background with neon accents, tech-noir style.
- **Prompt Veo 3.1:** The developer expands the holographic city with a hand gesture, data particles flying around, smooth camera orbit.
- **Narración:** "...es construir mundos. Es dar vida a la imaginación a través de la lógica."

## Escena 3: El Futuro es Ahora (12-20s)
- **Visual:** Un logo abstracto de una empresa de software emergiendo de una explosión de luz digital y circuitos.
- **Prompt Imagen 4:** Abstract tech logo made of glowing circuits and liquid metal, exploding with digital light particles, cyberpunk color palette (teal, magenta, gold), cinematic atmosphere.
- **Prompt Veo 3.1:** The logo stabilizes while the background continues to pulse with energy, slow zoom in.
- **Narración:** "Tu visión, codificada para el futuro. [Nombre de la Empresa / PoC]"
